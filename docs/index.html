Conversational Game Systems

AI DM Platform for Playable TTRPG Sessions

A product-first platform combining voice interaction, narrative AI, and top-down tactical gameplay into a single playable experience.

Pre-Alpha Voice + Text Input 2D Tactical Scenes Story-Pack Driven

What This Product Does

Conversational Game Systems is an AI-native software product focused on making tabletop role-playing sessions easier to start and easier to sustain. The platform runs AI game-master orchestration in real time, tracks structured world state, and switches to 2D battle views when encounters begin.

Core Product Modules

  • Real-time speech-to-speech and text interaction loop
  • LLM game-master orchestration and NPC response generation
  • Story-pack format for reusable adventures and world setups
  • State engine for quests, inventory, and encounter continuity
  • Top-down 2D tactical scene and battle visualization

Development Status

  • Defining the v1 story-pack schema and session state model
  • Building prototype voice pipeline for low-latency play
  • Implementing AI DM loop for narrative and combat turns
  • Testing tactical scene transitions for battle encounters

Who It Is For

  • Players who want AI-assisted solo or small-group campaigns
  • Teams that want fast start sessions with less prep overhead
  • Creators exploring AI-native interactive storytelling products
